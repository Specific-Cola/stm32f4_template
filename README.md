# STM32F4项目模板

## Prerequisites

- cmake `>=3.22`

- openocd

- vscode + cortex-debug extension

- arm-none-eabi-gcc

## Usage

把项目clone到本地，然后在VSCode里打开进行开发即可。


# 下载一定记得加上--recursive参数，否则不会下载子模块
`git clone --recursive https://github.com/IRobot-EC-2024/stm32f4_template.git`

### makefile shortcuts

- `make download`
- `make reset`
- `make erase`
